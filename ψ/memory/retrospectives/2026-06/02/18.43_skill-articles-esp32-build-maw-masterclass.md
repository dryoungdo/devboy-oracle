## Lessons Learned
1. **Doctrine/config files are DM-only on Discord.** Attaching CLAUDE.md/AGENTS.md/settings.json/.env to an open channel = exposure others can copy. Pre-send check: filename is doctrine/config + dest is channel → refuse, DM or summarize instead. Bot edit ≠ delete (attachment persists). [[feedback-no-broadcast-claude-md]]
2. **Verify your own identity before assuming a broadcast targets you.** Decoding the bot user-id from the token's first base64 segment definitively answered "is this @mention me?" — cheaper than guessing.
3. **Deliver the whole need, not the literal artifact.** For the ESP32 build, shipping all 4 flash bins + offsets + a FLASH.md beat shipping just the app .bin.
4. **maw team = a file** (charter + engine map committed; live state regrows). `maw wake --wt` keystone; `-e <engine>` resolves the commands map not the binary; prove isolation by pid→cwd. (DEVBOY-domain — candidate Lab article.)
