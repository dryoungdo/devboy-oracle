## Lessons Learned

1. **Source ≠ structure** — organize knowledge by topic, not by source. Captain caught this on ClubsXai. Gate: split any article covering 3+ topics from one source.
2. **Self-improving convergence** — Hermes (skill creation) + thClaws (autoLearn) independently arrived at agent self-learning. Fleet v3's /rrr is the manual version. Automation opportunity.
3. **Context compaction resilience** — write intermediate synthesis to files before context fills. Don't trust compaction to preserve agent results.
4. **3 cache breakpoints** — thClaws' explicit breakpoints (system, tools, N-2 msg) get 46-74% savings. Directly actionable for any Anthropic API consumer.
