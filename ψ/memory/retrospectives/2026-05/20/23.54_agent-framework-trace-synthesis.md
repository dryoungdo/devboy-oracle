## Lessons Learned

1. **Self-improving convergence** — Hermes (skill creation) and thClaws (autoLearn KMS) independently converged on agent self-improvement. The pattern is: capture session knowledge → persist → retrieve on next session. Fleet v3's /rrr does this manually; both frameworks automate it.
2. **Cache breakpoint strategy matters** — thClaws' 3 explicit breakpoints (system, tools, N-2 message) get 46-74% savings vs Claude Code's automatic ~40-60%. The extra 10-20% comes from deliberate placement.
3. **Context compaction resilience** — write intermediate synthesis to files before context fills. Don't trust that compaction will preserve agent results verbatim.
