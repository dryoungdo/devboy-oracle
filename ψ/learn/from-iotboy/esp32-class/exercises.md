# ESP32 Class — Exercise Log

Rolling log of practice exercises Captain works through, what worked, what drifted.

## Conventions

- **Newest first**, append at top
- One entry per exercise attempt — not per success. Failed attempts stay (Principle 1 — Nothing is Deleted)
- Include serial output snippet (timestamped) when relevant
- Tag the lecture: `[L01]`, `[L02]`, etc.

---

## Template

```
### YYYY-MM-DD — [Title]  [LNN]

**Goal**: what we tried to make happen
**Hardware**: ESP32 dev board, components, pin map
**Code**: link to `code/exercises/<slug>/` or paste sketch
**Result**: ✅ / ⚠️ / ❌
**Serial output** (relevant snippet):
```
[timestamp] message
```
**What I learned**:
- observation 1
- observation 2

**Next attempt**: what to change
```
