MCP Data Connectors

Claude เชื่อมต่อ institutional-grade data ผ่าน MCP:

  • FactSet, S&P Capital IQ, MSCI, PitchBook
  • Morningstar, LSEG, Daloopa, Dun & Bradstreet
  • IBISWorld, Third Bridge, Verisk
  • Third-party: EODHD MCP server, Alpha Vantage MCP

Excel & Google Sheets Integration

  • Claude available เป็น Office Add-in (Excel) และ Workspace Add-on (Google Sheets)
  • Sort/filter data, edit pivot tables, conditional formatting, data validation
  • DCF skill: build discounted cash flow model from scratch ใน Excel/Sheets
  • Variance analysis, budget vs actual, cash flow summaries, scenario narratives

How-To Workflow

  1. Install — add templates จาก anthropics/financial-services เป็น Claude Cowork plugin หรือ deploy via Managed Agents API
  2. Connect data — configure MCP connectors (FactSet, internal DBs, CSV/PDF uploads)
  3. Prompt with structure — ใช้ role-specific prompts ("You are a senior equity analyst...") + explicit output format
  4. Validate outputs — ทุก financial claim ต้อง cross-check. Never trust raw LLM output for numbers
  5. Audit trail — Claude generates full audit trails สำหรับ financial models

Model Performance

  • Opus 4.7: 64.37% Vals AI Finance Agent benchmark (SOTA)
  • FinanceBench: 82.7% financial document analysis
  • Hallucination rate: ~3% (vs ~6% GPT-5.5, ~6% Gemini 3.1 Pro)

Limitations & Risks

Hallucination in numbers: hallucinated account number หรือ fabricated regulatory citation = lawsuit. Always validate numerical outputs against source data.
  • Regulatory exposure — incorrect outputs อาจละเมิด SEC, GDPR, sector-specific regulations
  • Human-in-the-loop mandatory — Claude automates mechanical work; investment decisions ต้อง human review
  • Data freshness — real-time market data ต้อง live connectors (FactSet, MSCI)
  • Thailand BOT — FEAT principles (Fairness, Ethics, Accountability, Transparency) สำหรับ financial AI
