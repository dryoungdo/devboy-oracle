Antigravity คืออะไร

Google Antigravity คือ agent-first development platform ของ Google. เวอร์ชันแรก launch พ.ย. 2025 เป็น desktop editor app เดียว. ที่ Google I/O 2026 (19 พ.ค.) ขยายเป็น three-component stack: Desktop App + CLI + SDK

Antigravity CLI เป็น official successor ของ Gemini CLI — Gemini CLI จะหยุดให้บริการ 18 มิ.ย. 2026

Version 1.0 vs 2.0

Feature1.0 (พ.ย. 2025)2.0 (พ.ค. 2026)
ArchitectureSingle desktop editorDesktop App + CLI + SDK
Agent managementOne agent at a timeDynamic subagents (parallel)
Background tasksNoScheduled background tasks
Browser agentNoBuilt-in visual QA loop
VoiceNoNative voice commands
IntegrationsBasicAI Studio, Android, Firebase

CLI Features

  • Built in Go — เร็ว, resource overhead ต่ำ
  • Agent Skills — managed via /skills (file authoring หรือ npx skills install)
  • Hooks — JSON-defined lifecycle interceptors (before tool call, after file edit, on session start)
  • Subagents — spawn parallel work (refactors, research ทำ background ได้)
  • Plugins — Gemini CLI Extensions rebranded เป็น Antigravity plugins
  • Shared harness — Desktop app กับ CLI ใช้ agent harness เดียวกัน
