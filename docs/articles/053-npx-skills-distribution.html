npx skills — ติดตั้ง Skill ข้าม AI Agent

npx skills cross-agent arra-oracle-skills distribution 18-agents ✔ solid research

A — ปัญหา

Oracle มี skill ดีๆ เช่น /rrr, /learn, /dig — แต่ติดอยู่ใน Claude Code agent เดียว

ถ้าจะใช้ skill เดียวกันใน Codex, Cursor, Gemini CLI หรือ agent อื่น ต้อง copy SKILL.md ไปวางเอง — ทำไม่ scale

arra-oracle-skills-cli แก้ปัญหานี้ — คำสั่งเดียว ติดตั้ง skill ข้าม 18 AI agents

B — Quick Start

# ติดตั้ง standard profile ให้ Claude Code (global)
bunx --bun arra-oracle-skills install -g -y

# ติดตั้งให้หลาย agents พร้อมกัน
bunx --bun arra-oracle-skills install -g -y --agent claude-code codex opencode

# ติดตั้ง full profile
bunx --bun arra-oracle-skills install -g -y -p full

# เลือกเฉพาะ skill ที่ต้องการ
bunx --bun arra-oracle-skills install -g -y -s recap rrr trace

D — 4 Profile Tiers

ProfileSkillsใช้เมื่อ
minimal7มือใหม่ — about-oracle, forward-lite, go, recap-lite, rrr-lite, trace
standard13Daily driver — awaken, dig, forward, learn, recap, rrr, trace, xray + อื่นๆ

Profile selection มาจาก data — mine 120 sessions แล้วดูว่า skill ไหนถูกใช้ 10+ ครั้งถึงได้เข้า standard

E — ภายใน CLI ทำอะไร

E.1 — Skill Discovery

Skills อยู่ใน src/skills/ — แต่ละตัวมี SKILL.md + optional scripts/ + optional hooks

ใน compiled mode skills ถูก embed ใน VFS (Virtual Filesystem) ที่ generate ตอน build

E.2 — Install Process

  1. อ่าน profile → resolve skill list
  2. ตรวจ local-skill-precedence shield (#230) — skip skill ที่มี local version อยู่แล้ว
  3. Stamp SKILL.md ด้วย version + installer tag + scope (G-SKLL global, L-SKLL local)
  4. เขียน SKILL.md ไปยัง agent-specific path
  5. Hook-bearing skills → ติดตั้งเป็น Claude Code plugin ที่ ~/.claude/plugins/

E.3 — Installer Tag

ทุก SKILL.md ที่ติดตั้งจะมี marker installer: arra-oracle-skills-cli — ใช้สำหรับ safe auto-cleanup ของ orphaned skills

F — CLI Commands

Commandทำอะไร

G — Zombie Skills

บาง skill เป็น internal dev candidates — ไม่อยู่ใน profile ไหนเลย ติดตั้งได้เฉพาะ by name:

bunx --bun arra-oracle-skills install -s workon

ใช้สำหรับ skill ที่ยังไม่ stable พอจะ recommend ให้ทุกคน แต่ developer ที่รู้ว่าทำอะไรสามารถเรียกใช้ได้

H — Key Takeaways

  • คำสั่งเดียว ติดตั้ง skill ให้ 18 AI agents — ไม่ต้อง copy SKILL.md เอง
  • Profile system ป้องกัน information overload — มือใหม่ได้ 7 skills, power user ได้ 60+
  • Installer tag ทำให้ uninstall/update ปลอดภัย — ไม่แตะ skill ที่ user สร้างเอง
  • Local-skill-precedence — project-local skills ไม่โดน global overwrite
  • VFS + compiled binary — ไม่ต้อง clone repo, skills embed ใน CLI เลย
