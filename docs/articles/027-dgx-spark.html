DGX Spark คืออะไร

DGX Spark (เดิมชื่อ Project DIGITS) = desktop AI supercomputer ขนาด 1 ลิตร จาก NVIDIA — co-developed กับ MediaTek

NVIDIA positioning: "sovereign AI at desktop scale" — รัน model 200B parameters บนโต๊ะทำงาน data ไม่ออกจากเครื่อง
