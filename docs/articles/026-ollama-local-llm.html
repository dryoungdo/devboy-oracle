Ollama คืออะไร

Ollama = open-source local LLM runtime เขียนด้วย Go, wraps llama.cpp — "Docker for AI models"

Pull model ด้วยคำสั่งเดียว: ollama pull llama3 แล้วรันได้ทันที ไม่ต้อง config อะไร

Architecture

  • Model format: GGUF (binary, stores architecture + tokenizer + weights)
  • Default quantization: Q4_K_M (4-bit mixed-precision, ~75% compression vs FP16)
  • Modelfile: Dockerfile-like config — FROM, SYSTEM, TEMPLATE, PARAMETER
  • Storage: Content-addressable blob storage, layers shared across variants
  • Multi-model: 3+ models concurrent (controlled by OLLAMA_MAX_LOADED_MODELS)

Limitations

  • Not production-grade — ใช้ vLLM/SGLang สำหรับ production multi-user serving
  • Concurrency bottleneck — cross-model request queuing ทำให้ latency สูงเมื่อรัน multiple models
  • No auth/rate-limiting — API เปิด localhost ตรงๆ ต้องใส่ reverse proxy เอง
  • ROCm fragility — AMD GPU บน Linux ยังมีปัญหา driver version mismatch
