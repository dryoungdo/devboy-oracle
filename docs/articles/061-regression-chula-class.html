1 — Regression คืออะไร?

ML มี 2 งานหลักใน Supervised Learning:

งานคำถามผลลัพธ์ตัวอย่าง
Classificationอยู่กลุ่มไหน?ป้ายชื่อ (label)spam / not spam, แมว / หมา
Regressionเท่าไหร่?ตัวเลข (number)ราคาบ้าน, อุณหภูมิพรุ่งนี้, ยอดขาย

จำง่าย: Classification = "ถังไหน?" — Regression = "เท่าไหร่?"

2.3 — สูตรเส้นตรง: y = mx + c

สัญลักษณ์ชื่อความหมาย
yผลลัพธ์ราคาบ้าน (สิ่งที่ต้องการทำนาย)
xinputขนาดบ้าน (ตร.ม.)
mความชัน (slope)ราคาเพิ่มขึ้นกี่ล้าน ต่อ ตร.ม. ที่เพิ่ม
cจุดตัดแกน (intercept / bias)ราคาฐานตั้งต้น (ถ้าบ้านมีขนาด 0)

2.4 — หาค่า m และ c จากข้อมูล

ใช้ Least Squares (หาเส้นที่ทำให้ระยะห่างรวมจากจุดทั้ง 5 ไปถึงเส้นน้อยที่สุด):

ข้อมูล 5 จุด:
  x = [30, 50, 70, 90, 120]      y = [1.5, 2.5, 3.2, 4.0, 5.5]

ขั้นที่ 1: หาค่าเฉลี่ย
  x̄ = (30+50+70+90+120) / 5 = 360/5 = 72
  ȳ = (1.5+2.5+3.2+4.0+5.5) / 5 = 16.7/5 = 3.34

ขั้นที่ 2: คำนวณ m (slope)
  m = ∑(x-x̄)(y-ȳ) / ∑(x-x̄)²

  แต่ละจุด:
  (30-72)(1.5-3.34) = (-42)(-1.84) =  77.28
  (50-72)(2.5-3.34) = (-22)(-0.84) =  18.48
  (70-72)(3.2-3.34) = (-2)(-0.14)  =   0.28
  (90-72)(4.0-3.34) = (18)(0.66)   =  11.88
  (120-72)(5.5-3.34)= (48)(2.16)   = 103.68
                                     -------
  ∑(x-x̄)(y-ȳ) = 211.60

  (30-72)²  = 1764
  (50-72)²  =  484
  (70-72)²  =    4
  (90-72)²  =  324
  (120-72)² = 2304
              ----
  ∑(x-x̄)² = 4880

  m = 211.60 / 4880 = 0.04336 ล้าน/ตร.ม.
    ≈ 43,360 บาท ต่อ ตร.ม.

ขั้นที่ 3: คำนวณ c (intercept)
  c = ȳ - m·x̄
  c = 3.34 - 0.04336 × 72
  c = 3.34 - 3.122 = 0.218 ล้าน

สูตรที่ได้:

ราคา = 0.0434 × ขนาด + 0.218

(หรือ: ราคา = 43,360 × ตร.ม. + 218,000 บาท)

2.5 — ทำนาย: บ้าน 80 ตร.ม. ราคาเท่าไหร่?

ราคา = 0.0434 × 80 + 0.218
     = 3.472 + 0.218
     = 3.69 ล้านบาท

3 — "Training" คืออะไร?

Training = หาค่า m กับ c ที่ทำให้เส้นอยู่ใกล้จุดทั้งหมดมากที่สุด

ลองนึกภาพ: ถ้าเราลากเส้นมั่ว ๆ จุดจะห่างจากเส้นเยอะ ถ้าเราปรับ m กับ c ทีละนิด เส้นจะค่อย ๆ เข้าใกล้จุดมากขึ้น — นั่นคือ "การเรียนรู้"

เส้นแย่ (m, c มั่ว):          เส้นดี (m, c ที่ train แล้ว):

  *                              *
       *                      . *
  --------  เส้นห่างจุด     . *
     *                    .*
  *                     .*
                       .  ← จุดอยู่ชิดเส้น!

4 — วัดว่าเส้นดีแค่ไหน (Error)

ถ้าจุดจริงคือ 3.2 ล้าน แต่เส้นทำนาย 3.0 ล้าน → error = 0.2 ล้าน

เราเอา error ของทุกจุดมาคิดรวม:

จุด      จริง    ทำนาย    ห่าง     ห่าง²
30 ตร.ม.  1.50    1.52    -0.02     0.0004
50 ตร.ม.  2.50    2.39     0.11     0.0121
70 ตร.ม.  3.20    3.26    -0.06     0.0036
90 ตร.ม.  4.00    4.12    -0.12     0.0144
120 ตร.ม. 5.50    5.43     0.07     0.0049
                                   -------
            Mean Squared Error =    0.0354/5
                               =    0.0071

RMSE = √0.0071 = 0.084 ล้าน ≈ 84,000 บาท

แปลว่า: โดยเฉลี่ย เส้นของเราทำนายคลาดเคลื่อนประมาณ 84,000 บาท — สำหรับบ้านหลักล้าน ถือว่าดี

5 — Multiple Features: ราคาบ้านไม่ได้ขึ้นกับขนาดอย่างเดียว

ความจริงแล้ว ราคาบ้านขึ้นกับหลายปัจจัย:

  • ขนาด (ตร.ม.)
  • จำนวนห้องนอน
  • คะแนนทำเล (0-10)
  • อายุบ้าน (ปี)

สูตรขยายจาก y = mx + c เป็น:

1 feature:    ราคา = m·ขนาด + c

4 features:   ราคา = w1·ขนาด + w2·ห้องนอน + w3·ทำเล + w4·อายุ + bias

ตัวอย่างค่าที่ train ได้:
  ราคา = 0.043·ขนาด + 0.5·ห้องนอน + 0.3·ทำเล - 0.05·อายุ + 0.2

  w1 = +0.043  → ขนาดเพิ่ม 1 ตร.ม. ราคาขึ้น 43,000
  w2 = +0.5    → ห้องนอนเพิ่ม 1 ห้อง ราคาขึ้น 500,000
  w3 = +0.3    → ทำเลดีขึ้น 1 คะแนน ราคาขึ้น 300,000
  w4 = -0.05   → บ้านเก่าขึ้น 1 ปี ราคาลด 50,000
  bias = 0.2   → ราคาฐาน 200,000

สิ่งที่เปลี่ยน: แทนที่จะมี m ตัวเดียว ตอนนี้มี w (weight) หลายตัว — ตัวละปัจจัย — บวก bias 1 ตัว

Training = หาค่า w1, w2, w3, w4, bias ที่ทำให้ error น้อยที่สุด — หลักการเดิม

6 — สูตรนี้ คือ Neuron! (Mind-Blown)

กลับไปดูสูตร multiple regression:

y = w1·x1 + w2·x2 + w3·x3 + w4·x4 + bias

ตอนนี้วาด neuron 1 ตัว:

        weights
x1 —— w1 —╮
x2 —— w2 —╮
               ∑ + bias —→ output (y)
x3 —— w3 —╮
x4 —— w4 —╮

Neuron ทำอะไร?
  1. รับ inputs (x1, x2, x3, x4)
  2. คูณแต่ละตัวด้วย weight (w1, w2, w3, w4)
  3. บวกทั้งหมดเข้าด้วยกัน + bias
  4. ได้ output

→ สูตร เดียวกันกับ multiple linear regression!
ภาษา Regressionภาษา Neural Network
feature (x)input
weight / slope (w)weight (w)
intercept (c)bias (b)
prediction (y)output
training (หา w, c ที่ดีที่สุด)training (ปรับ weights ให้ดีที่สุด)

Linear Regression = Neuron 1 ตัว ที่ไม่มี activation function

พอเอา neuron หลายตัวมาต่อกันเป็น layer → ANN → ต่อหลาย layer → Deep Learning

Linear Regression        1 Neuron              ANN/DNN
y = wx + b          →  y = f(wx + b)     →  หลาย neurons + หลาย layers
                         f = activation         = Deep Learning
                         function

ทุกอย่างเริ่มจากเส้นตรงเส้นเดียว y = mx + c

7 — Beyond Linear: เส้นโค้ง (Polynomial Regression)

ถ้าข้อมูลไม่เป็นเส้นตรง เราใช้ polynomial:

Linear (degree 1):     y = w·x + b                เส้นตรง
Quadratic (degree 2):  y = w1·x² + w2·x + b      โค้ง 1 ที
Cubic (degree 3):      y = w1·x³ + w2·x² + w3·x + b  โค้ง S

ข้อควรระวัง: degree สูง = เส้นพลิ้วตาม training data → overfitting

Degree 1 (เส้นตรง):       Degree 10 (โค้งมั่ว):
                           ∾    ∾
 *  .····  *              * ∽  ∽  *
.··´       .·*          ∾ ∾     ∾ ∾
                         ∽*         *∽
อาจพลาดบ้าง              จำ data เก่าแม่น
แต่ทำนาย data ใหม่ได้ดี   แต่ทำนาย data ใหม่แย่มาก

เหมือน Classification ที่ decision boundary ซับซ้อนเกินไป (ดู article 057) — ปัญหาเดียวกัน ชื่อเดียวกัน: overfitting

สรุป

#บทเรียน
1Regression = ทำนายตัวเลข (≠ Classification ที่ทำนายกลุ่ม)
2y = mx + c — m บอกว่า input เปลี่ยน 1 หน่วย output เปลี่ยนเท่าไหร่, c คือค่าตั้งต้น
3Training = หาค่า m, c ที่ดีที่สุด ให้เส้นอยู่ใกล้ข้อมูลจริงมากที่สุด
4Error วัดเป็นตัวเลข — RMSE บอกว่าโดยเฉลี่ยทำนายพลาดเท่าไหร่
5Multiple features → weight หลายตัว + bias — สูตรเดียวกับ neuron 1 ตัว
6Neuron = Linear Regression + activation function — ต่อหลายตัว = ANN = Deep Learning
7Polynomial = เส้นโค้ง — ยืดหยุ่นกว่า แต่ degree สูงเกิน = overfitting
