  1. Opt-in: autoLearn: true in settings
  2. Session end → KmsCreate bootstraps dedicated self_learn KMS
  3. /kms ingest <kms> $ — files session content เป็น one page
  4. Throttled /kms reconcile — dedup contradictions

3 Anthropic Cache Breakpoints (46–74% savings)

thClaws ใส่ cache_control: ephemeral ที่ 3 จุดใน AnthropicProvider::build_body:

  1. System prompt — byte-stable จนกว่า CLAUDE.md/memory/KMS เปลี่ยน
  2. Last tool definition — covers entire tools array; เปลี่ยนเมื่อ MCP connect/disconnect หรือ skill install
  3. Second-to-last message — rolling conversation breakpoint; index N-2 byte-stable (N-1 = live user turn)
ScenarioSavings
10 turns, normal coding~46%
10 turns, tool-heavy~54%
30 turns, normal coding~74%

Source: thclaws-technical-manual/prompt-cache.md

AGENTS.md (Vendor-Neutral Standard)

thClaws รองรับทั้ง AGENTS.md (vendor-neutral, Agentic AI Foundation) และ CLAUDE.md (Anthropic):

Load order (later overrides earlier):

  1. User-global: ~/.claude/{CLAUDE,AGENTS}.md + ~/.config/thclaws/AGENTS.md
  2. Walk up from cwd: CLAUDE.md + AGENTS.md in each ancestor
  3. Project dirs: .claude/CLAUDE.md, .thclaws/{CLAUDE,AGENTS}.md
  4. Rules dirs: .{claude,thclaws}/rules/*.md (alphabetically)
  5. Local overrides: CLAUDE.local.md, AGENTS.local.md (highest priority)

Quick Start

# Install from releases
# https://github.com/thClaws/thClaws/releases

# Build from source (Rust 1.85+, Node 20+, pnpm 9+)
git clone https://github.com/thClaws/thClaws.git
cd thClaws
cd frontend && pnpm install && pnpm build && cd ..
cargo build --release --features gui --bin thclaws

# Run
thclaws              # GUI
thclaws --cli        # CLI REPL
thclaws -p "prompt"  # one-shot

# Configure
/provider anthropic
/model claude-sonnet-4-6
