thClaws — Open-Source Agent Harness Platform

thclaws rust multi-provider agent · 20 พ.ค. 2026 · ❓ raw

thClaws คืออะไร

thClaws คือ open-source AI agent workspace เขียนด้วย native Rust — ทำงาน local บนเครื่องตัวเอง "Sovereign by design"

เป็นทั้ง Claude Code competitor และ multi-provider agent platform — 948 stars, 134 forks, Apache-2.0 license

4 Surfaces, 1 Engine

Agent loop, Session, ToolRegistry เดียวกันทุก UX:

  • Desktop GUI (thclaws) — native window: Terminal + Chat + Files + Team tabs
  • CLI REPL (thclaws --cli) — interactive terminal สำหรับ SSH/headless
  • Non-interactive (thclaws -p "prompt") — single turn แล้วจบ สำหรับ scripts/CI
  • Webapp (thclaws --serve --port 7878) — WebSocket/HTTP ผ่าน SSH tunnel

Multi-Provider Support

รองรับ 300+ models ผ่าน OpenRouter + native providers:

  • Anthropic — Claude (ทุก version) + Claude Agent SDK
  • OpenAI — Chat Completions + Responses/Codex
  • Google — Gemini & Gemma
  • Thai LLMs — OpenThaiGPT, Typhoon, Pathumma, THaLLE (NSTDA)
  • Others — DeepSeek, Qwen, NVIDIA NIM, Azure AI, Ollama (local)
  • Generic — OpenAI-compatible slot (oai/*) สำหรับ LiteLLM, vLLM, proxies

Key Features

  • Skills — reusable workflows packaged เป็น directory + SKILL.md
  • MCP servers — third-party tools + OAuth 2.1 + PKCE
  • KMS (Knowledge Base) — per-project wiki, grep + read (no embeddings), /dream consolidation
  • Agent Teams — concurrent agents ใน tmux panes + git worktrees + shared mailbox
  • Plan Mode — sequential multi-step + user approval gates
  • Document Workflow — native PDF, DOCX, PPTX, XLSX tools
  • Schedule — cron jobs + intervals + filesystem watchers + launchd/systemd daemon
  • Hooks — pre/post tool_use, permission_denied, session_start, pre_compact
  • Memory — AGENTS.md / CLAUDE.md compatible project instructions

Quick Start

# Install from releases
# https://github.com/thClaws/thClaws/releases

# หรือ build from source (Rust 1.85+, Node 20+, pnpm 9+)
git clone https://github.com/thClaws/thClaws.git
cd thClaws
cd frontend && pnpm install && pnpm build && cd ..
cargo build --release --features gui --bin thclaws

# Run
thclaws              # GUI
thclaws --cli        # CLI REPL
thclaws -p "prompt"  # one-shot

# Configure provider
/provider anthropic
/model claude-sonnet-4-6
