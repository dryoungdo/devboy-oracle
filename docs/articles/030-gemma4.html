Gemma 4 คืออะไร

Gemma 4 = Google DeepMind's open-weight model family — released 2 เม.ย. 2026 — Apache 2.0 (เปลี่ยนจาก custom license ใน v1-3)

Built from same research as Gemini 3 — แต่เปิดให้ใช้ commercial ได้เต็มที่ ไม่มี MAU restrictions

Architecture

  • Transformer + Grouped Query Attention (GQA)
  • SiGLU activation
  • Dual RoPE (standard + p-RoPE) for extended context
  • Multimodal: text + image input ทุก size | audio + video input สำหรับ E2B/E4B
  • Text output only
