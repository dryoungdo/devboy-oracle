maw-ui — ARRA Office Fleet Dashboard

maw-ui คืออะไร

เชื่อมต่อกับ maw-js ยังไง

  • WebSocket feed จาก maw-js backend port :3456 — real-time, ไม่ poll
  • ?host=<peer> query param บนทุกหน้า — ชี้ไปที่ maw-js node ไหนก็ได้
  • Peer execution — POST /api/peer/exec ส่ง signed command relay
  • Peer proxy — REST relay สำหรับ HTTP-LAN peers

Shape A Deployment (แนะนำ)

# Production — Shape A (one port, one process)
maw ui --install
# downloads dist → ~/.maw/ui/dist/
# maw-js serves ที่ :3456

# Dev mode
npm install
npm run dev
# Vite HMR on :5173, proxy /api + /ws → localhost:3456

# Cloudflare Workers
npx wrangler deploy --config wrangler.god.json

# Hosted version
# https://god.buildwithoracle.com/federation_2d?host=<your-node>
