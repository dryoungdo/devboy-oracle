Gemini Spark คืออะไร

Gemini Spark (ไม่ใช่ "Sparks") คือ persistent agentic AI personal assistant จาก Google ประกาศที่ Google I/O 2026 (19 พ.ค. 2026). Sundar Pichai อธิบายว่า “your personal AI agent that helps you navigate your digital life, taking action on your behalf and under your direction.”

ต่างจาก chatbot ทั่วไป — Spark รัน 24/7 บน dedicated cloud VMs ใน Google Cloud ทำงานต่อแม้ปิดมือถือหรือ laptop

Architecture

ComponentDetail
Base modelGemini 3.5 Flash (4x faster, 76.2% Terminal-Bench 2.1, 83.6% MCP Atlas)
Agent harnessAntigravity 2.0 — goal persistence, task decomposition, tool orchestration, safety constraints, state recovery
InfrastructureDedicated Google Cloud VMs per user, always-on
Safety layerThree-layer: model + harness constraints + explicit user authorization for high-stakes actions

Key Features

  • Always-on background operation — ทำงานขณะคุณนอน
  • Google Workspace integration — Gmail, Calendar, Drive, Docs, Sheets, Slides, YouTube, Maps (out of the box)
  • Dedicated Gmail address — email Spark ได้ตรง
  • Chrome integration — interact กับ web ผ่าน Chrome (coming later 2026)
  • Proactive tasks — parse credit card statements, monitor school emails, create daily digests, turn meeting notes into polished Docs
  • Custom sub-agents — coming later 2026
  • Third-party — Canva, OpenTable, Instacart, Uber, Zillow (coming months)
  • Mac app — standalone Gemini Mac app with local file access (summer 2026)
  • MCP support — adopted at I/O 2026, Linux Foundation standard
