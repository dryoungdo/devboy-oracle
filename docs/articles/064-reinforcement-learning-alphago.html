Supervised Learning = มีครูบอกคำตอบ, Unsupervised = ไม่มีคำตอบหาเอง, แล้ว Reinforcement Learning ล่ะ? = ลองทำ → ได้ feedback → ปรับ → ทำใหม่

1 — เหมือนฝึกหมา

ลองนึกภาพ:

  • หมานั่ง → ได้ขนม (+1 reward)
  • หมากระโดดขึ้นโต๊ะ → ถูกดุ (-1 punishment)
  • ทำซ้ำร้อยครั้ง → หมาเรียนรู้: นั่ง = ดี, กระโดด = ไม่ดี

ไม่มีใคร "สอน" หมาด้วยตำรา ไม่มี label ไม่มีเฉลย — แค่ ลองทำ + ได้ feedback + ปรับพฤติกรรม

2 — 5 คำที่อธิบายทุกอย่าง

คำความหมายตัวอย่าง: หมาตัวอย่าง: AlphaGo
Agentผู้เรียนหมาโปรแกรม AI
Environmentโลกที่มันอยู่บ้านกระดาน Go 19×19
Stateสถานการณ์ตอนนี้"อยู่ใกล้โต๊ะ เจ้าของมองอยู่"หินทุกตัวบนกระดาน
Actionทำอะไรได้บ้างนั่ง, กระโดด, เห่าวางหินที่ตำแหน่ง X
Rewardคะแนน feedback+1 ขนม, -1 ถูกดุ+1 ชนะ, -1 แพ้

Agent ดู state → เลือก action → ได้ reward → เห็น state ใหม่ → ทำซ้ำเป็นล้านครั้ง → ค่อยๆ สร้าง "แผนที่ภายใน" ว่าสถานการณ์ไหนควรทำอะไร

3 — เรื่อง AlphaGo: Timeline จริง

ตุลาคม 2015 — ชนะครั้งแรก

AlphaGo (Google DeepMind) ชนะ Fan Hui แชมป์ Go ยุโรป 5-0 — AI ตัวแรกที่ชนะผู้เล่นมืออาชีพ

ขั้นตอนการเรียนรู้:

  1. Supervised Learning — ศึกษาจาก 30 ล้านตาเดินของผู้เล่นมืออาชีพ
  2. Reinforcement Learning — แข่งกับตัวเองเป็นล้านเกม (self-play)

มีนาคม 2016 — แมตช์ที่เปลี่ยนโลก

AlphaGo vs Lee Sedol (ตำนาน Go ระดับโลก) ที่กรุงโซล — คนดูกว่า 200 ล้านคน

ผล: AlphaGo ชนะ 4-1

เกม 2 — AlphaGo เดิน "Move 37" — ตาเดินที่แปลกมากจนนักวิเคราะห์คิดว่าผิดพลาด (โอกาสเกิดในเกมมนุษย์ = 1 ใน 10,000) — แต่กลับเป็นตาที่ยอดเยี่ยม

Lee Sedol ชนะเกม 4 — ครั้งเดียวที่มนุษย์ชนะ AlphaGo

พฤษภาคม 2017 — Ke Jie (อันดับ 1 โลก) แพ้ 0-3

DeepMind ให้ AlphaGo เกษียณจากการแข่งขัน

ตุลาคม 2017 — AlphaGo Zero: เริ่มจากศูนย์

AlphaGo Zero ไม่ใช้ข้อมูลมนุษย์เลย — รู้แค่กฎ + เล่นกับตัวเอง

เวลาผลลัพธ์
3 วัน (4.9 ล้านเกม)ชนะ AlphaGo เวอร์ชัน Lee Sedol 100-0
21 วันเทียบเท่า AlphaGo Master (ตัวที่ชนะ Ke Jie)
40 วันเก่งกว่าทุกเวอร์ชันก่อนหน้า

บทเรียนสำคัญ: เรียนรู้จากมนุษย์ = เรียนรู้ bias ของมนุษย์ด้วย — เริ่มจากศูนย์ = ค้นพบกลยุทธ์ที่มนุษย์ไม่เคยคิด

4 — Exploration vs Exploitation: ร้านอาหาร

Exploitation = ทำสิ่งที่รู้แล้วว่าดี — ไปร้านประจำทุกศุกร์ อร่อยแน่นอน

Exploration = ลองสิ่งใหม่ — ไปร้านที่ไม่เคยไป อาจอร่อยกว่า... หรืออาจแย่มาก

ถ้า exploit อย่างเดียว  → ไม่เคยค้นพบร้านที่ดีกว่า
ถ้า explore อย่างเดียว  → เสียเวลากับร้านแย่ๆ ไม่ได้กินร้านที่ชอบ
สมดุล                   → กินร้านประจำ + ลองร้านใหม่บ้าง = ดีที่สุด

AlphaGo Zero ตอนแรกลองเดินแปลกๆ เยอะมาก (explore) → พอเก่งขึ้น เน้น exploit กลยุทธ์ดีๆ + ยังลองของใหม่บ้าง → Move 37 คือผลจาก balance นี้
