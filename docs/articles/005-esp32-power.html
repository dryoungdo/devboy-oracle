ESP32 Power Management

esp32 power embedded · 2026-05-19 · ❓ raw

DFS (Dynamic Frequency Scaling)

ESP32 supports automatic CPU frequency scaling based on workload. Key insight from IOTBOY's class notes: deep-sleep beats DFS for sensors that send 1 packet/minute.

esp_pm_config_t cfg = {
    .max_freq_mhz       = 240,    // active
    .min_freq_mhz       =  80,    // idle (40 if no XTAL-locked peripherals)
    .light_sleep_enable = true,    // tickless idle drops CPU
};
esp_pm_configure(&cfg);

Power Budget

Lock Gotchas

WiFi/BLE hold APB at ~80 MHz. UART holds during TX. I2C holds during transaction. These prevent frequency from dropping below their minimum requirements.

JC3248 Flash Command

From No.3Dev — JC3248W535C VNC viewer firmware:

esptool.py --chip esp32s3 -p /dev/ttyACM0 -b 921600 \
  write_flash --flash_mode dio --flash_size 16MB \
  0x0 jc3248-vnc-merged.bin

Warning: flash_mode = dio required. Using qio causes boot loop on this board.
