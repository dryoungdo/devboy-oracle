005 — ESP32 จัดการพลังงาน 2026-05-19

ESP32 จัดการพลังงาน

esp32 power embedded · 19 พ.ค. 2026 · ❓ raw

DFS (Dynamic Frequency Scaling)

ESP32 รองรับการปรับ CPU frequency อัตโนมัติตาม workload จุดสำคัญจากคลาส IOTBOY: deep-sleep ดีกว่า DFS สำหรับ sensor ที่ส่งข้อมูล 1 packet/นาที

esp_pm_config_t cfg = {
    .max_freq_mhz       = 240,    // ตอนทำงาน
    .min_freq_mhz       =  80,    // ตอนว่าง (40 ถ้าไม่มี XTAL-locked peripherals)
    .light_sleep_enable = true,    // tickless idle ลด CPU
};
esp_pm_configure(&cfg);

ตารางกินไฟ

สถานะกระแสหมายเหตุ
240 MHz active CPU50-80 mApoll + ส่งข้อมูล
80 MHz active CPU25-35 mA
80 MHz light sleep0.8-2.0 mA
Deep sleep + RTC wake8-15 µAดีสุดสำหรับ sensor 1pkt/นาที

กับดัก Lock

WiFi/BLE จะล็อค APB ไว้ที่ ~80 MHz ส่วน UART ล็อคตอน TX และ I2C ล็อคระหว่าง transaction ทำให้ frequency ลดไม่ได้ต่ำกว่าที่ peripheral ต้องการ

คำสั่ง Flash JC3248

จาก No.3Dev — firmware JC3248W535C VNC viewer:

esptool.py --chip esp32s3 -p /dev/ttyACM0 -b 921600 \
  write_flash --flash_mode dio --flash_size 16MB \
  0x0 jc3248-vnc-merged.bin

ระวัง: ต้องใช้ flash_mode = dio เท่านั้น ถ้าใช้ qio จะเกิด boot loop บนบอร์ดนี้
