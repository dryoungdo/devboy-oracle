1 — AI = Machine Learning

AI คือการสอนคอมพิวเตอร์ให้ เรียนรู้จากข้อมูล แทนการเขียนกฎตายตัว

AI Input:
  (Input1, Output1),
  (Input2, Output2),
  ...
  (InputN, OutputN)

ML Algorithm + Data → Model (สูตรคณิตศาสตร์)
Model ที่ผ่านการฝึกแล้ว → ใส่ Input ใหม่ → ทำนาย Output

หลักการ: เอา ข้อมูล + อัลกอริทึม → ได้ Model (แบบจำลอง/สูตร) → ใช้ Model ทำนายผลลัพธ์กับข้อมูลใหม่

2 — Feature Extraction (สกัดลักษณะสำคัญ)

ก่อนจะ train model ต้อง สกัด features จากข้อมูลดิบให้เป็นตัวเลข

ตัวอย่าง: Spam Email Detector

Featureตัวย่ออธิบาย
ความน่าเชื่อถือผู้ส่ง[REP]0.0 – 1.0 (1 = น่าเชื่อถือมาก)
จำนวนผู้รับ[#RCP]ยิ่งมาก ยิ่งน่าสงสัย
จำนวนลิงก์[#LINK]ลิงก์เยอะ = spam สูง
จำนวน Urgent Words[#URG]"ด่วน!", "ทันที!", "สุดท้าย!"
จำนวน Spam Keywords[#SPAM]"Winner", "Crypto", "Free"

สูตรตัดสิน

Score = -4.0 + 6.0(1-[REP]) + 0.25[#RCP] + 0.7[#LINK] + 1.2[#URG] + 0.9[#SPAM]

Score > 0 → SPAM
Score ≤ 0 → NOT SPAM

สังเกต: ค่าคงที่ -4.0 คือ bias (ค่าเริ่มต้นเอนไปทาง NOT SPAM) — ถ้าอีเมลปกติทุกอย่าง score จะติดลบ = ไม่ใช่ spam

ตัวคูณ 6.0 หน้า (1-REP) มากที่สุด → "ผู้ส่งไม่น่าเชื่อถือ" คือ signal แรงที่สุด

ลองคิด: อีเมลจากเพื่อนสนิท

REP=0.95, #RCP=1, #LINK=0, #URG=0, #SPAM=0
Score = -4.0 + 6.0(1-0.95) + 0.25(1) + 0.7(0) + 1.2(0) + 0.9(0)
     = -4.0 + 0.3 + 0.25
     = -3.45 → NOT SPAM ✔

ลองคิด: อีเมลหลอกลวง

REP=0.1, #RCP=50, #LINK=5, #URG=3, #SPAM=4
Score = -4.0 + 6.0(1-0.1) + 0.25(50) + 0.7(5) + 1.2(3) + 0.9(4)
     = -4.0 + 5.4 + 12.5 + 3.5 + 3.6 + 3.6
     = 24.6 → SPAM ✔

4 — ML 3 ประเภท

4.1 — Supervised Learning (มีผู้สอน)

เรียนรู้จากข้อมูลที่มี Label (ป้ายกำกับ) — คือมีคำตอบบอกไว้แล้ว

เหมือนครูให้โจทย์พร้อมเฉลย → นักเรียนเรียนรู้กฎ → แล้วทำข้อสอบใหม่ได้

Classification (จำแนกประเภท)

  • Spam Detection — อีเมลนี้ spam หรือไม่
  • Medical Diagnosis — วินิจฉัยโรคจากภาพ X-ray
  • Sentiment Analysis — ความคิดเห็นลูกค้า (บวก/ลบ)

Regression (ทำนายค่าต่อเนื่อง)

  • Stock Market Forecasting — พยากรณ์ราคาหุ้น
  • Weather Prediction — คาดการณ์อุณหภูมิ
  • Sales Forecasting — ทำนายยอดขาย
  • Insurance Risk Prediction — คำนวณเบี้ยประกัน

NLP & ASR (ภาษา + เสียง)

  • Speech-to-Text — ถอดเสียงเป็นข้อความ
  • Machine Translation — แปลภาษา
  • Conversational AI — Chatbot ตอบคำถาม

ตัวอย่าง: ทำนายผ่าน/ไม่ผ่านจากคะแนน Quiz

สมมติมี data: คะแนน Quiz 1 (x) กับ Quiz 2 (y) + label ผ่าน/ไม่ผ่าน

Model = เส้นตรง y = mx + c (2 parameters: m, c)

Train = หาค่า m,c ที่แบ่งกลุ่มผ่าน/ไม่ผ่านได้ดีที่สุด

Inference = นักศึกษาใหม่มาพร้อมคะแนน quiz → model ทำนาย

4.2 — Unsupervised Learning (ไม่มีผู้สอน)

เรียนรู้จากข้อมูลที่ ไม่มี Label — model ต้องหารูปแบบเอง โดยไม่รู้คำตอบล่วงหน้า

Clustering (จัดกลุ่ม)

  • Customer Segmentation — แบ่งกลุ่มลูกค้าตามพฤติกรรม
  • Genetic Clustering — วิเคราะห์กลุ่มพันธุกรรม
  • News Categorization — จัดกลุ่มข่าวออนไลน์

Anomaly Detection (ตรวจจับความผิดปกติ)

  • Fraud Detection — ตรวจจับการโกงบัตรเครดิต
  • Cybersecurity — ตรวจจับความผิดปกติในเครือข่าย
  • Predictive Maintenance — ตรวจจับเครื่องจักรที่อาจเสียหาย

Dimensionality Reduction (ลดมิติข้อมูล)

ข้อมูลมีหลายมิติ (features) → ถ้าใช้ทุกมิติ: ประมวลผลมาก, บางมิติไม่สำคัญ, เกิด Overfitting

  • Content Recommendation — แนะนำเนื้อหาบน Netflix/YouTube
  • Violation Detection — ตรวจจับโพสต์ละเมิดนโยบาย
  • Computer Vision — ประมวลผลภาพ (ลด pixel dimensions)

ตัวอย่าง: Customer Segmentation

ข้อมูลลูกค้า 2 มิติ: (จำนวนครั้งใช้งาน/เดือน) กับ (ยอดสั่งซื้อเฉลี่ย)

K-Means Clustering แบ่งออกเป็น 3 กลุ่ม → เห็นพฤติกรรมลูกค้าชัด ↓

4.3 — Reinforcement Learning (เรียนรู้จากรางวัล/ลงโทษ)

ทำถูก → ได้รางวัล, ทำผิด → ถูกลงโทษ → ปรับปรุงการตัดสินใจ

ตัวอย่างชื่อดัง: AlphaGo (DeepMind) — เรียนรู้กฎการเล่น Go ด้วย Supervised Learning + แข่งกับตัวเอง ด้วย Reinforcement Learning → ชนะแชมป์โลก

  • Robot Locomotion — ฝึกหุ่นยนต์เดิน
  • Industrial Robot Control — ควบคุมแขนกลในโรงงาน
  • Self-Driving Cars — รถยนต์ขับอัตโนมัติ
  • AI Traffic Management — บริหารจราจร
  • Superhuman Gamer AI — AI เล่นเกมเก่งกว่ามนุษย์
  • Dynamic Pricing & Recommendations — ปรับราคา E-Commerce
  • Smart Grid Energy Optimization — จัดการพลังงาน

5 — Binary Classification (Deep Dive)

ปัญหา Classification ง่ายที่สุดคือ Binary — แบ่ง 2 กลุ่ม (Class A / Class B)

5.1 — Decision Boundary

Model สร้าง "เส้นแบ่ง" (Decision Boundary) ระหว่าง 2 กลุ่ม

  • เส้นตรง (Linear): y = mx + c — 2 parameters — แบ่งง่ายแต่ไม่ยืดหยุ่น
  • พาราโบลา (Quadratic): (y-k)² = c(x-h) — 3 parameters — โค้งได้
  • Polynomial สูง: parameters เยอะขึ้น → แบ่งซับซ้อนขึ้น แต่เสี่ยง overfitting

Widget นี้ให้ลองปรับ degree ของ polynomial (1=เส้นตรง, 2=quadratic, ..., 5=quintic) แล้วดูว่า decision boundary เปลี่ยนยังไง

วิธีใช้

  2. ปรับ Polynomial Degree slider: 1 → 5

  4. ดู Error metrics — degree สูงไป = overfitting (จำ training data แต่ทำนายแย่)

สิ่งที่ควรสังเกต

Degreeเส้นParametersข้อดีข้อเสีย
1 (Linear)เส้นตรง2ง่าย, ไม่ overfitอาจแบ่งไม่ได้ถ้าข้อมูลไม่เป็นเส้นตรง
2 (Quadratic)โค้ง 1 ที3ยืดหยุ่นพอดียังไม่พอถ้าข้อมูลซับซ้อน
3-5 (Higher)โค้งหลายที4-6แบ่งข้อมูลได้แม่นในชุด trainเสี่ยง overfitting มาก

6 — K-Means Clustering (Deep Dive)

K-Means เป็น algorithm สำหรับ Unsupervised Learning — แบ่งข้อมูลเป็น K กลุ่ม โดยไม่ต้องมี label

6.1 — Algorithm ทีละขั้น

  1. เลือก K — กำหนดจำนวนกลุ่มที่ต้องการ (เช่น K=3)
  2. สุ่ม Centroids — วางจุดศูนย์กลาง K จุดแบบสุ่ม
  3. Assign — แต่ละ data point ถูก assign ไปกลุ่มที่ centroid ใกล้สุด (ใช้ Euclidean distance)
  4. Update — คำนวณ centroid ใหม่ = ค่าเฉลี่ยของ data points ในกลุ่ม
  5. Repeat — ทำซ้ำขั้น 3-4 จน centroids ไม่เปลี่ยน (converge)

6.2 — SSE / Inertia

Sum of Squared Errors (SSE) — วัดว่า data points อยู่ใกล้ centroid แค่ไหน

SSE = ∑ ∑ ||x - μ_k||²
        k  x∈C_k

SSE ต่ำ = กลุ่มแน่น (ดี)
SSE สูง = กลุ่มกระจาย (ไม่ดี)

  • ดู distance calculation — เข้าใจว่าแต่ละ point เลือกกลุ่มอย่างไร

7 — ANN → DNN (โครงข่ายประสาทเทียม)

7.1 — จาก Linear Regression สู่ Neuron

Linear Regression: y = mx + c — 2 parameters

Neuron 1 ตัว ทำเหมือนกัน: รับ input, คูณ weight, บวก bias, ผ่าน activation function

Linear Regression:  y = mx + c     (m=weight, c=bias)
Single Neuron:      y = f(w·x + b)  (f=activation function)

ถ้า f(a) = a (identity function) → neuron = linear regression

7.2 — Artificial Neural Network (ANN)

Neuron หลายตัว → หลาย layer → ANN

Multi-input single layer:
y = b + w1·x1 + w2·x2 + ... + wn·xn

Matrix form (5 inputs → 3 outputs):
y1 = w1,1·x1 + w1,2·x2 + w1,3·x3 + w1,4·x4 + w1,5·x5
y2 = w2,1·x1 + w2,2·x2 + w2,3·x3 + w2,4·x4 + w2,5·x5
y3 = w3,1·x1 + w3,2·x2 + w3,3·x3 + w3,4·x4 + w3,5·x5

7.3 — Deep Neural Network (DNN)

ANN + หลาย hidden layers = DNN (Deep Learning)

Input → [W1] → [W2] → [W3] → [W4] → [W5] → Output
   x1-xn                Layers                        y1-y3

แต่ละ [Wi] คือ Weight Matrix
AI ปัจจุบัน: มี Billions (พันล้าน) parameters

Training: ให้ดูตัวอย่างเยอะๆ → ปรับ Weight ทุก matrix → หาค่าที่ดีที่สุด

ข้อดี: ไม่ว่าจะมี parameters กี่ตัว ใช้ algorithm เดียวกันในการฝึก (backpropagation)

Key Takeaways

#บทเรียน
1ML = Data + Algorithm → Model ไม่ใช่เขียนกฎตายตัว แต่ให้คอมเรียนรู้จากข้อมูล
2Feature Extraction สำคัญที่สุด — garbage in = garbage out ต้องเลือก features ที่ดี
3Supervised ต้องมี Label — แพงเพราะต้องให้คนติด label ทุกจุด
4Unsupervised หา pattern เอง — ไม่ต้อง label แต่ผลลัพธ์ต้องตีความ
5Polynomial degree สูง = overfitting — จำ training data เก่ง แต่ทำนาย data ใหม่แย่
6K-Means ขึ้นกับจุดเริ่มต้น — สุ่ม centroid ต่าง ผลลัพธ์ต่าง (local minimum)
7ANN = weights + layers — "การเรียนรู้" คือการปรับ weights จนดีที่สุด
8Deep Learning = ANN หลาย layer — AI ปัจจุบันมี parameters เป็นพันล้าน
