4. Quick Start (5 นาที)

  1. Install plugin: claude plugin install commercial-legal@claude-for-legal
  2. Upload non-sensitive contract (vendor NDA, open-source license)
  3. Run: /contract-review
  4. Review output — treat เหมือน junior associate's draft
  5. Verify every citation กับ Westlaw/LexisNexis ก่อนใช้
Citation hallucination = non-negotiable risk. LLM citation hallucination rates: 58–88% depending on model and court hierarchy. Latham & Watkins (Anthropic's own counsel) ถูกตำหนิในศาลเพราะ Claude สร้าง citation ปลอม. ต้อง verify ทุก citation กับ primary source เสมอ.

8. API & Agent Patterns

Contract Review Agent (Python)

import anthropic

client = anthropic.Anthropic()

def review_contract(contract_text, jurisdiction="California"):
    response = client.messages.create(
        model="claude-opus-4-7-20250506",
        max_tokens=8192,
        system=f"""You are a contract review assistant for
{jurisdiction} law. Analyze each clause for:
1. Risk level (H/M/L)
2. Market-standard assessment
3. Suggested edits
4. Relevant statutory/case law

NEVER fabricate citations. If unsure, state explicitly.""",
        messages=[{
            "role": "user",
            "content": f"Review this contract:\n\n{contract_text}"
        }]
    )
    return response.content[0].text

11. Limitations & Risks

Legal AI failures are malpractice-shaped. A fabricated case citation in a brief is a Rule 3.3 violation. A missed liability cap in a contract review is negligence. These are not theoretical risks.
  • Citation hallucination: 58-88% of AI-generated legal citations are fabricated or inaccurate. Mata v. Avianca (S.D.N.Y. 2023) = $5,000 sanctions for fake citations
  • Jurisdiction confusion: Claude may conflate state laws, apply wrong version of statute, or cite repealed provisions
  • Confidentiality: client data in prompts = potential privilege waiver on non-enterprise tiers
  • Recency: knowledge cutoff means recent case law/statutes may be missing or outdated
  • Multi-jurisdictional: cross-border analysis accuracy drops significantly — verify each jurisdiction separately
  • Unauthorized practice: AI output to clients without attorney review may constitute UPL
