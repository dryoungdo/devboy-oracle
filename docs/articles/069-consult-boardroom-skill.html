A — มันคืออะไร: ไม่ใช่ “ถาม AI” แต่เป็น “นั่งหัวโต๊ะประชุม”

ปกติถาม AI เราได้คำตอบเดียว เกลี่ย ๆ ปลอดภัย. /consult เปลี่ยนเกม — มันจำลอง boardroom ที่ คุณคือประธาน แล้วเรียกที่ปรึกษา 2–4 คนมาเถียงกันต่อหน้า: พูดกับคุณตรง ๆ ("you"), ตอบโต้กันเอง เรียกชื่อกัน, ขัดจังหวะ, ท้าทาย, มี stage direction (*leans forward*, *shakes head*) แล้วปิดด้วยสรุปว่าตรงไหนเห็นตรงกัน ตรงไหนแตก.

Mental model: การตัดสินใจเชิงกลยุทธ์ไม่ได้ต้องการ “ความเห็นผู้เชี่ยวชาญ” เดี่ยว ๆ — มันต้องการ ความตึง (tension) ระหว่างมุมที่ขัดกัน. skill นี้ขาย tension เป็นฟีเจอร์ ไม่ใช่ bug.

Bilingual จริง (ไม่ใช่แค่แปล): ถามไทยตอบไทย ถามอังกฤษตอบอังกฤษ — แต่ signature quote คงภาษาอังกฤษเดิมเสมอ. trigger ทั้งสองภาษา เช่น /consult ควรเขียน auth ใหม่ทั้งหมดไหม?

B — กลไกลับ: lazy-load + “tension injection table”

2 อย่างที่ทำให้ skill นี้ฉลากกว่าที่คิด:

  • Lazy-load: ไม่ได้โหลด persona ทั้ง 36 ไฟล์ (= 80K+ tokens). SKILL.md เก็บแค่ ตารางดัชนี เล็ก ๆ (36 แถว keyword) → เลือก 2–4 คนที่ match → อ่านเฉพาะไฟล์ที่เลือกจาก disk. budget เหลือ ~9K–23K tokens ต่อ session.
  • Tension injection: มีตารางความขัดแย้งที่ตั้งใจ — ถ้าคำถามแตะหัวข้อที่รู้ว่ามีศึก จะโหลดทั้งสองฝั่งเข้ามาบังคับให้เถียง. ตัวอย่างคลาสสิก — "ควร rewrite ไหม?":
Rewrites tension: Torvalds (อย่า rewrite เด็ดขาด) ↔ Hotz (rewrite เลย ลบทิ้ง) ↔ Carmack (profile ก่อน วัดก่อนค่อยตัดสิน) — โหลดมาครบ 3 คนเพื่อกัน “เออออเห็นด้วยหมด”.

⇒ มันไม่ปล่อยให้ความขัดแย้งเกิดเอง (emergent) แต่ ออกแบบความขัดแย้งไว้ล่วงหน้า. นี่คือเหตุผลที่ debate มันคม ไม่ใช่ AI ตีกินปลอม ๆ.

G — Verdict: ควร install ไหม?

✅ ควร install — skill เฉพาะทางคุณภาพสูง, ติดตั้ง 0 ต้นทุน

install ฟรี ไม่มี dependency ลบทิ้งได้ทุกเมื่อ — downside แทบเป็นศูนย์. upside สูงถ้าคำถามตรงกับจุดแข็งของมัน. สำหรับคนไทยยิ่งคุ้ม: roster ที่ปรึกษาไทย (ตัน/ท็อป/กระทิง/ธนินท์) ลึกและ research จริง — น่าจะเป็น skill ที่ปรึกษากลยุทธ์ไทยที่ดีที่สุดเท่าที่มี.

✅ ใช้ตอนไหน❌ ไม่ควรใช้ตอนไหน
คำถามกลยุทธ์ที่มี “ศึก” ชัด (rewrite vs ไม่, ระดมทุน vs bootstrap, concentrate vs diversify) งาน 1 คำตอบถูก (fix TypeScript error, ถาม return type) — skill skip เองอยู่แล้ว
คำถามตลาด/ธุรกิจไทย (สร้างแบรนด์, startup ไทย vs ต่างประเทศ) ต้องการคำตอบใน 10 วินาที — มัน verbose by design
สถาปัตยกรรม engineering (abstract เพิ่ม vs keep simple) อยากได้ consensus/ความกลมเกลียว — มันจงใจ inject ความขัดแย้ง
เบื่อ AI ที่ตอบปลอดภัยเกลี่ย ๆ อยากได้ Taleb เถียง Buffett จริง ๆ หัวข้อนอก 11 โดเมนที่ครอบคลุม (HR, วัฒนธรรม, soft-skill ล้วน)
